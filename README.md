## Project Description

This repository contains the implementation of **Lab 02 – Introduction to HTML & GitHub Pages**.
The objective of this lab is to understand the fundamental structure of HTML documents and learn how to create and publish web pages using GitHub Pages.

The project demonstrates the use of core HTML elements such as text formatting, hyperlinks, images, tables, forms, and semantic navigation structures.

---

## Learning Objectives

Through this lab, the following concepts were practiced:

* Understanding the basic structure of an HTML5 document
* Creating and formatting text using HTML tags
* Working with hyperlinks and images
* Creating tables for structured data
* Designing forms for user input
* Implementing navigation between multiple web pages
* Deploying static websites using GitHub Pages

---

## Project Files

The repository contains the following HTML pages:

* **index.html** – Main landing page of the website
* **profile.html** – Personal profile page with basic information
* **schedule.html** – Weekly course schedule represented using an HTML table
* **navigation.html** – Navigation menu linking all pages
* **registration.html** – Student registration form using HTML form elements
